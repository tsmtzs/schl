\begin{schltask}
\item \points{25}\\
  Κάθε μία από τις παρακάτω προτάσεις είναι είτε σωστή ( Σ ), είτε λάθος ( Λ ). Να κυκλώσετε τη σωστή απάντηση.
  \begin{truefalse}
  \item Η παράγωγος μιας συνάρτησης $f$ στο $x_0$, εκφράζει το  ρυθμό  μεταβολής του $y = f(x)$ ως προς το $x$, όταν $x = x_0$.
  \item Ισχύει ότι $ (\cos x)' = -\sin x $.
  \item Ισχύει ότι $(e^x)' = e^x$.
  \item Ισχύει ότι $ \bigl( f(x)\cdot g(x) \bigr)' = f'(x)\cdot g(x) + f(x)\cdot g'(x) $.
  \item Ισχύει ότι $ \biggl( f\bigl( g(x) \bigr) \biggr) ' = f' \bigl( g'(x) \bigr) $.
  \end{truefalse}
\item \points{25}\\
  Να αποδείξετε ότι για την παράγωγο της σταθερής  συνάρτησης $f(x) = c$ ισχύει  $(c)' = 0$.
\item \points{25}\\
  Δίνεται η συνάρτηση $f(x) = x^2 - 4$, όπου $x\in\mathbb{R}$. Να υπολογίσετε το όριο
  $\limdisplay{x\to-3} \frac{f(x) - 5}{x+3}$.
  % Use \leavevmode to start a paragraph and then go to next line.
  % Without \leavevmode the next code throws an error.
\item \leavevmode\\ Θεωρούμε τη συνάρτηση $f(x) = x e^x$, όπου $x\in\mathbb{R}$.
\begin{enumerate}[leftmargin=14pt]
 \item Να δείξετε ότι η συνάρτηση $f$ ικανοποιεί τη σχέση $f''(x) - f(x) = 2 e^x$.\points{10}
 \item Να βρείτε την εξίσωση της εφαπτομένης της γραφικής παράστασης της $f$ στο σημείο με τετμημένη $x_0 = 0$.\\\phantom{.}\points{15}
 \end{enumerate}
\end{schltask}
